OH-TECH Guide

使い方

このサイトは、情報Ⅰの４択問題を解きながら、６分野ごとの進み具合・正答率・苦手問題を確認できる学習サイトです。初回に学年・組・番号を登録すると、学習結果は終了時に自動送信されます。

１．最初に行うこと

  1. トップ画面の「学習記録の送信設定」で、学年・組・番号を入力します。
  2. 「登録・更新」を押すと、この端末に設定が保存されます。
  3. 設定後は、問題を解き終えたときに学習結果が自動で送信されます。

未設定のままでも練習はできますが、授業で記録を確認する場合は、先に登録してください。

２．６つの分野

  • 情報社会と法：情報モラル、個人情報、著作権、産業財産権など
  • 情報デザイン：見やすさ、伝わりやすさ、ユニバーサルデザインなど
  • コンピュータの仕組み：２進数、デジタル表現、CPU、記憶装置、OSなど
  • プログラミング：アルゴリズム、フローチャート、擬似言語、処理の流れなど
  • ネットワークとセキュリティ：通信の仕組み、暗号化、認証、マルウェア対策など
  • データ活用：表・グラフ、統計、相関、回帰、データ分析など

３．通常の学習を始める

  1. トップ画面の「トレーニング設定」で、分野・難易度・問題数を選びます。
  2. 計算問題を重点的に練習したい場合は「計算問題のみ」を選びます。
  3. 計算問題を避けたい場合は「計算問題を除外」を選びます。
  4. 「スタート」を押すと、条件に合う問題がランダムに出題されます。

４．トップ画面の見方

  • 今日の学習状況：今日解いた問題数や正答率を確認できます。
  • 次にやること：続きから、苦手問題、未学習問題などをすぐに始められます。
  • 分野別進捗度：６分野ごとの学習済み数、未学習数、達成率、正答率を確認できます。

未学習が多い分野や正答率が低い分野から取り組むと、弱点を減らしやすくなります。

５．問題一覧・記録確認

  1. トップ右上の「問題一覧・記録を見る」を開きます。
  2. 分野・難易度・学習状態・計算問題・キーワードで絞り込みます。
  3. 問題は10件ずつ表示されます。10件を超える場合は「前の10件」「次の10件」でページを切り替えます。
  4. 「表示中をすべて選択」は、現在表示されている10件だけを選択します。
  5. 解きたい問題を選んで「選んだ問題を開始」を押すと、選択した問題だけを練習できます。

６．苦手問題の考え方

  • 直近誤答：最後に解いたときに間違えた問題です。
  • 弱点問題：これまでの正答率が70％未満の問題です。
  • 勘で解いた問題：自己チェックで「勘」を選んだ問題です。
  • 自信なし問題：自己チェックで「不正解」を選んだ問題です。

復習では、直近誤答 → 弱点問題 → 勘で解いた問題の順に確認すると効果的です。

７．解答後の自己チェック

  • 自信あり：理由まで理解して解けたときに選びます。
  • 勘：正解したが、考え方に自信がないときに選びます。
  • 不正解：間違えた、または解説を読んでも不安が残るときに選びます。

自己チェックを正直に選ぶほど、復習するべき問題が見つけやすくなります。

８．メダルの条件

  • 🥇 金メダル：その問題の直近10回の正答率が90％以上
  • 🥈 銀メダル：その問題の直近10回の正答率が70％以上
  • 🥉 銅メダル：その問題の直近10回の正答率が50％以上
  • 10回未満の問題は、メダル判定の準備中として扱います。

９．おすすめの学習順

  1. まずは６分野の未学習問題を少しずつ減らします。
  2. 正答率が低い分野は、初級・基礎から解き直します。
  3. 直近誤答や「勘」で解いた問題を復習します。
  4. 計算が苦手な場合は「計算問題のみ」で短時間練習します。
  5. テスト前は、メダルがない問題や銅メダル以下の問題を重点的に確認します。

この端末のブラウザにも学習記録が保存されます。問題ごとの履歴は分析用に直近30回分、苦手問題リストは最大100問分を保持します。別の端末やブラウザではローカル記録は共有されません。学習記録の送信設定をしている場合は、学習終了時に結果が自動送信されます。
