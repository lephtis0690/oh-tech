# answersシート対応 実装報告

## 実装内容

学習終了時に送信される学習記録データに、1問ごとの詳細ログ `answers` が含まれることを確認しました。

`answers` には次の内容が含まれます。

- 問題ID
- 分野
- 生徒が選択した選択肢
- 正解の選択肢
- 正誤
- 自信度（サイト内データとして保持）

## スプレッドシート側で追加するシート

`answers` シートを作成し、1行目に次の見出しを設定してください。

```text
日時 / 生徒ID / 学年 / 組 / 番号 / 問題ID / 分野 / 選択肢 / 正解 / 正誤
```

## Apps Script

`docs/apps-script-learning-log-with-answers.js` に、`sessions` と `answers` の両方へ記録するApps Scriptコードを同梱しました。

既存のApps Scriptをこの内容に差し替え、保存後に「デプロイを管理」から新しいバージョンとして再デプロイしてください。

## 補足

サイト側の送信先URLは既存のApps Script URLを維持しています。
